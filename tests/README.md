# Testes do Projeto

Esta pasta contém os documentos relacionados aos testes realizados no sistema IMMA Atacadista.

Os testes foram elaborados com foco na validação das funcionalidades principais do MVP, garantindo o correto funcionamento das regras de negócio e da autenticação do sistema.

---

## Conteúdo

- Relatório de testes
- Casos de teste
- Resultados esperados e obtidos

---

## Funcionalidades Testadas

- Login de usuários
- Validação de campos obrigatórios
- Navegação entre telas
- Cadastro de usuários
- Tratamento de mensagens de erro

---

## Objetivo

Garantir estabilidade, confiabilidade e funcionamento correto das principais funcionalidades do sistema.
