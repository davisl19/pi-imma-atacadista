# Instalação e Execução

Este documento descreve os passos necessários para executar o projeto localmente.

---

## Requisitos

Antes de iniciar, é necessário possuir instalado:

- Flutter SDK
- Android Studio
- Git
- Emulador Android ou dispositivo físico conectado

---
## Clonando o Repositório

```bash
git clone https://github.com/davisl19/pi-imma-atacadista.git
```

---

## Acessando a Pasta do Projeto

```bash
cd pi-imma-atacadista/mobile
```

---

## Instalando Dependências

```bash
flutter pub get
```

---

## Executando o Projeto

```bash
flutter run
```
