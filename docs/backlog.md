# Backlog Inicial – Épicos

Este documento apresenta o backlog inicial do sistema proposto para a IMMA Atacadista, contendo os épicos e suas respectivas user stories.

---

## Épico 1 — Inteligência Logística

**Descrição:**  
Criação do módulo responsável por organizar pedidos automaticamente para otimizar o processo logístico e facilitar a montagem das cargas de entrega.

### User Stories

- **US01 — Agrupamento de pedidos por cidade**  
  *Como responsável pela logística, quero que os pedidos sejam agrupados automaticamente por cidade para facilitar o planejamento das rotas de entrega.*

- **US02 — Geração automática de romaneios**  
  *Como operador logístico, quero gerar automaticamente romaneios de carga para agilizar o processo de separação e carregamento.*

- **US03 — Visualização das cargas organizadas**  
  *Como operador logístico, quero visualizar os pedidos organizados por carga para acompanhar o planejamento das entregas.*

---

## Épico 2 — Gestão de Recebíveis (Malote)

**Descrição:**  
Desenvolvimento do módulo responsável pela conferência digital dos documentos de recebimento e controle dos pagamentos realizados pelos clientes.

### User Stories

- **US04 — Controle de status de pagamento**  
  *Como responsável financeiro, quero marcar pedidos como pagos ou em vale para acompanhar a situação financeira dos clientes.*

- **US05 — Alertas de vencimento**  
  *Como responsável financeiro, quero receber alertas de vencimentos próximos para evitar atrasos nos recebimentos.*

---

## Épico 3 — Dashboard de Performance

**Descrição:**  
Criação de um painel de visualização de dados que permita acompanhar o desempenho de vendas e o cálculo automático das comissões.

### User Stories

- **US06 — Visualização de desempenho de vendas**  
  *Como vendedor, quero visualizar meu desempenho de vendas em um painel para acompanhar meus resultados.*

- **US07 — Cálculo automático de comissões**  
  *Como vendedor, quero que minhas comissões sejam calculadas automaticamente para evitar erros de cálculo.*
  
---

## Épico 4 — Integração ERP

**Descrição:**  
Desenvolvimento da integração entre o sistema e o ERP TopGerente para garantir sincronização de dados entre as plataformas.

### User Stories

- **US08 — Sincronização de pedidos com ERP**  
  *Como sistema, quero enviar automaticamente os pedidos registrados para o ERP TopGerente para manter os dados atualizados.*

- **US09 — Importação de dados do ERP**  
  *Como sistema, quero importar dados de clientes e produtos do ERP para garantir que as informações estejam atualizadas.*

- **US10 — Atualização em tempo real**  
  *Como gestor, quero que as informações sejam sincronizadas em tempo real entre os sistemas para evitar divergências.*
