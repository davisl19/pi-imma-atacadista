# Backend

Esta pasta será destinada ao desenvolvimento do backend do sistema.

## Objetivos do Backend

- Processamento e estruturação de dados
- Integração com IA Generativa
- Comunicação com o ERP TopGerente
- Aplicação das regras de negócio
- Sincronização de pedidos e recebíveis

## Tecnologias Previstas

- Python
- FastAPI
- APIs de IA Generativa

## Situação Atual

Nesta etapa do projeto, o backend ainda se encontra em fase de planejamento e definição arquitetural.

A estrutura foi mantida no repositório para representar os componentes futuros responsáveis pela lógica de negócio, integração com IA e comunicação com o ERP.
